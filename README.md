# Data Engineering Projects (Python)

This repository contains hands-on data engineering projects implemented
using Python. The projects are designed to build strong fundamentals and
progress gradually from beginner to advanced-level data engineering concepts.

---

## Projects

### 1. ETL Project using OOP (Core Python)
**Level:** Beginner  
**Description:**
- Extracts data from CSV / Json files
- Applies data validation, transformation, and business rules using OOP principles
- Loads processed data into target files and PostgreSQL database
- Focuses on clean code, modular design, and reusability

**Concepts Covered:**
- Object-Oriented Programming (OOP)
- ETL pipeline design
- Error handling & logging
- Code structuring for data pipelines
